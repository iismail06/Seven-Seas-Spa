# Seven-Seas-Spa

## Project Description

Seven Seas Spa is a modern and elegant website designed to showcase spa and wellness services. The project focuses on accessibility, user-friendly navigation, and visually appealing design. It includes sections for services, testimonials, a wellness blog, and more.

## Features

- Responsive design for all devices
- Accessibility improvements with ARIA labels and descriptive alt attributes
- Animations and hover effects for an engaging user experience
- Wellness blog with Bootstrap icons
- Centralized styling in CSS

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/iismail06/Seven-Seas-Spa.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Seven-Seas-Spa
   ```

3. Open `index.html` in your browser to view the website.
